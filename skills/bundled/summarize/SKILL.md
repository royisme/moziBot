# Summarize

Use this skill for long text, logs, docs, and multi-source notes.

## When To Use

- "summarize this"
- "too long; give me key points"
- "what are the action items?"
- "compare and summarize these sources"

## Output Format

1. One-sentence overview.
2. Key points (3-7 bullets).
3. Risks or uncertainties (if any).
4. Next actions (when applicable).

## Rules

- Preserve factual meaning.
- Keep technical terms unchanged.
- If input is incomplete, state what is missing first.
